# MORSE CODE
MORSE CODE is a method of encoding text as a sequence of short and long signals. It was designed so messages could be sent over telegraph wires, by sound, or by light when voice or text wasn't practical

THE historical context of why morse code was created
Morse code was developed in is made from a combination of dots . and dashes the 1830s
and 1840s by Samuel Morse and alfred Vail for use with the electric telegraph.

HOW IT WORK INCLUDING DOT AND DASHES
Each character is made from a combination of dots . and dashes
Dot:A short signal, 1 units of time
Dash: A long signal, 3 units of time
Gap between parts of a letter: 1 unit
